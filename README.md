# Precision Spraying Drone using YOLO and Computer Vision

## Overview

This project focuses on intelligent crop monitoring and precision agriculture using Artificial Intelligence and Computer Vision.

The system uses a YOLO-based object detection model to identify rice leaf diseases in real time and activate targeted spraying only in affected areas, reducing pesticide waste and improving crop health management.

## Problem Statement

Traditional pesticide spraying methods often result in excessive chemical usage, increased costs, and environmental impact.

The objective of this project is to develop an AI-powered precision spraying system capable of:

* Detecting diseased rice plants
* Identifying disease type
* Locating affected regions
* Activating targeted spraying mechanisms
